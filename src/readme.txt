The following third party code is used for this implementation:

LeafletJS
=========
leaflet.zip (Leaflet 1.0.1)
Available online from: http://leafletjs.com/download.html

LeafletJS - Leaflet.markercluster - Marker clustering plugin
https://github.com/Leaflet/Leaflet.markercluster

Leaflet.GridLayer.GoogleMutant
https://gitlab.com/IvanSanchez/Leaflet.GridLayer.GoogleMutant

Leaflet.fullscreen - fullscreen control
Leaflet.fullscreen-gh-pages.zip (https://github.com/Leaflet/Leaflet.fullscreen/archive/gh-pages.zip)
https://github.com/Leaflet/Leaflet.fullscreen

LeafletJS - Leaflet.Spin - spinner plugin 
Leaflet.Spin-master.zip (https://github.com/makinacorpus/Leaflet.Spin/archive/master.zip)
https://github.com/makinacorpus/Leaflet.Spin

SpinJS - Javascript spinner
spin.js
spin.min.js
http://fgnass.github.io/spin.js/

Leaflet.Sleep - deactivates the map so that it doesn't zoom when you scroll a page containing the map
Leaflet.Sleep.js
https://github.com/CliffCloud/Leaflet.Sleep
